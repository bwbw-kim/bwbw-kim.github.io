ABOUT ME



안녕하세요 저는 성균관대학교 반도체시스템공학을 전공하고


현재는 삼성전자에서 소프트웨어엔지니어로 재직중인 김병욱입니다.


사람 만나는 것을 좋아하고 워낙 호기심이 많아서


항상 새로운걸 시도하고 또 도전하는 삶을 살고있습니다.




PORTFOLIO



- Code Review Bot

형상관리시스템에서 Pull Request(PR)를 생성하면, 자동으로 리뷰 코멘트를 달아주는 코드 리뷰 시스템을 Python으로 개발하였습니다

이 시스템은 Jenkins CI/CD 툴로 사용하여 구현하였습니다.



- Unit Test Platform

Unit Test의 중요성을 회사 내에 홍보하기 위해 여러 가지 시스템을 구축하였습니다

이 과정에서, Django를 사용하여 형상관리시스템들의 Webhook을 조작할 수 있는 시스템을 개발하였으며

Springboot로 여러 개의 형상관리시스템 API를 통합하는 시스템을 개발하였습니다.



- DSDN (Samsung Stackoverflow)

사내 환경에서 개발 시 발생하는 어려움을 해결하기 위해 Q&A 플랫폼을 개발하는 프로젝트에 참여하였습니다

이 프로젝트는 Springboot를 기반으로 하였으며, 프론트엔드부터 백엔드까지 모든 개발 단계에 참여하였습니다.
