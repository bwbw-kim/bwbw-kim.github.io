PORTFOLIO



- Code Review Bot

코드리뷰 효율화를 위해 만들어져 형상관리시스템의 PR에 자동으로 리뷰 코멘트를 달아주는 시스템 개발에 참여했습니다.

Python 으로 개발 되었으며 CI/CD tool 인 Jenkins 를 통하여 동작하도록 구현했습니다.



- Unit Test Platform

Unit Test 의 중요함을 사내에 전파하기 위해 여러가지 시스템을 구축하였습니다.

그 과정중 하나로 형상관리시스템들의 Webhook을 자유자재로 조작 할 수 있는 Django로 제작한 시스템 개발에 참여하였고

여러개의 형상관리시스템들의 api를 하나로 통합한 Springboot로 제작한 시스템 개발에 참여하였습니다.



- DSDN (Samsung Stackoverflow)

사내환경에서 개발시 어려움을 해결하기위한 Q&A 플랫폼 제작에 참여했습니다.

Springboot 기반으로 개발 되었으며 frontend 부터 backend 까지의 개발 전부에 참여하였습니다.
